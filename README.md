Aplicacion movil realizada en android studio para el proyecto de aplicaciones moviles

Esta aplicacion integrada con firebase utilizando realtime database y auth para la creacion y login de usuarios en la app permite utilizar 
un porton automatico de manera remota. 
La aplicacion permite abrir/cerrar el porton a traves de un servomotor, registrar los valores de los sensores LDR y de distancia, configurar el umbral de deteccion ademas de
la configuracion de los horarios de cierre o apertura y los horarios para encender o apagar un led. 
Cuenta tambien con un control de alertas y una configuracion de modos tanto automatico, manual y de desactivacion del sistema.

Esta aplicacion se ha realizado en conjunto a un proyecto arduino para el funcionamiento de un porton automatico.
